Privacy Policy for AI Reading Assistant

Last updated: May 17, 2026

What data does this extension collect?

This extension does NOT collect, store, or transmit any personal data to any third-party server.

All data stays on your device:

  • API Key: Stored locally using Chrome's storage.local API. Never sent anywhere except directly to the AI provider (OpenAI or Anthropic) for authentication.
  • Selected text: The text you select is sent directly from your browser to the AI provider's API (OpenAI or Anthropic) for processing. It is never sent to any server we control.
  • Usage counter: A daily count of API calls is stored locally in storage.local for the free usage limit feature. This never leaves your device.

How is data transmitted?

The extension makes API calls directly from your browser to the AI provider you configure (OpenAI at api.openai.com or Anthropic at api.anthropic.com). The only data transmitted is:

  • Your API key (sent as an HTTP header to authenticate with the AI provider)
  • The text you selected (sent as part of the API request to generate the summary/explanation/translation)

These transmissions are encrypted via HTTPS.

Third-party services

The extension uses the following third-party APIs, which have their own privacy policies:

  • OpenAI API — Privacy Policy
  • Anthropic API — Privacy Policy

Do we track users?

No. This extension has:

  • No analytics
  • No cookies
  • No fingerprinting
  • No advertising SDKs
  • No data collection of any kind
